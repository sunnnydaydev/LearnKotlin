
# 一 、基本语法
###### 1、函数的定义
方式1（常见方式）：
```kotlin
语法:
fun 函数名字（参数1：参数类型，参数2：参数类型........）：函数返回值类型{
         // 函数体
}
栗子（两数字之和）：
fun sum(a: Int, b: Int): Int {
    return a + b
}
```

方式2 (表达式作为函数体，返回值类型自推断)
```kotlin

  栗子：
  fun sum（a：Int，b:Int） = a+b

```
方式3（无返回值类型 Unit）
```kotlin
语法：
       fun 函数名（参数：参数类型，xxx，xxx   .....）：Unit（Unit关键字可无）{
         // 方法体
       }

 栗子：
 fun sum(a: Int, b: Int):Unit{//:unit 可省略
     println(a+b)
}

```
